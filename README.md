PROYECTO DE PATRIA A PATRIA

En el proyecto de patria a patria encontraras una descripción de un viaje de uno los estudiantes de TripleTen desde Kentucky a Burundi pasando por Gales y Ucrania.

Lo más importante de este proyecto es la versatilidad del mismo, aquí podrás observarlo desde un smartphone hasta tu computadora personal, siendo el mismo creado con un diseño web adaptativo. Siéntete libre de probar la funcionalidad del mismo en diferentes pantallas

visita el proyecto a través de GitHub pages con el siguiente enlace: https://ericksj91.github.io/web_project_homeland/
